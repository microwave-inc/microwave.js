Config.json layout:

```json
{
    "token": "TOKEN HERE",
    "prefix": "m!",
    "activity": [
        "Insert activity here"
    ],
    "clientId": "CLIENT ID",
    "guildId": "GUILD ID"
}
```

Make sure to rename `example.config.json` to `config.json` and fill everything in... ClientID and GuildID are for deploying commands
